# 📈 Stock Market Prediction Dashboard

An AI-powered stock forecasting dashboard that combines **Machine Learning**, **financial time-series analysis**, and **interactive visualization** to predict stock price trends and evaluate forecasting performance.

Built as part of my application project for **Uneeq Interns**.

---

## 🚀 Overview

This project allows users to:

* Enter a stock ticker symbol
* Select a historical date range
* Train a forecasting model
* Generate future price predictions
* Compare actual vs predicted values
* Visualize training performance and forecasting metrics

The goal of this project was not only building a prediction model, but also creating a dashboard experience that makes forecasting more understandable and interactive.

## ⚙️ Project Workflow

1. Fetch historical stock data
2. Clean and preprocess data
3. Normalize values
4. Create sliding windows
5. Train forecasting model
6. Generate predictions
7. Evaluate results
8. Display analytics dashboard

## ▶️ Installation

Clone the repository:

```bash
git clone YOUR_REPOSITORY_LINK
```

Go to project directory:

```bash
cd StockMarketPrediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```plaintext
http://localhost:5000
```
